Quickstart Guide¶

In our quickstart guide we briefly describe how our program can be used. For further information read our documentation.

Table of content:

  1. Requirements
  2. Start the program
  3. Train a network
  4. Classify a repository

Requirements¶

  1. numpy
  2. tensorflow
  3. gensim
  4. PyQt5
  5. requests_oauthlib
  6. requests

Start the program¶

With our console navigate into our project directory. Then start the userinterface with the following command:

python3 gui.py

Use the program to train a neural network¶

When you started the program a GUI will show. Click on the tab training. Then select the appropriate files and directories to run the training.

Pick the listed files and directories in the same order as shown here:

  1. Unclassified training data: (Choose multiple file) Every File in the data/json directory.
  2. Main directory of the network: Take a directory of your choice.
  3. Sub directory of the network: Take a directory of your choice.
  4. Validationset data: Select the file validationData.json.
  5. Validation data labels: Select the file validationDataLabels.json.
  6. Supervised trainset data: Select every file in tagged/data.
  7. Supervised trainset labels: Select every file in tagged/labels.

Use the program to classify a repository of your choice¶

If you want to classify a repository start the program follow these steps.

  1. Create a file with all repository URLs you want to classify. For each repository use just one line.
  2. Click the button ‘login to GitHub’.
  3. The browser will open GitHub automatically. Log in.
  4. The URL in your browser will change. Copy the new URL.
  5. Click the ‘login’ button of our GUI.
  6. Then click the button ‘classify repositories’
  7. Wait till the program finishes working then see the results in the results tab.
